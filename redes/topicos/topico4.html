Tipos de Rede

27 de Abril, 2017.

Uma rede é uma coleção de computadores que se comunicam entre si, afim de compartilhar informações, funcionalidades e serviços, como por exemplo dados, softwares e hardwares. De forma sucinta, vamos a alguns exemplos de tipo de rede:

- LAN (Local Area Network): A LAN é uma das redes mais comuns; ela conecta computadores, impressoras e outros dispositivos dentro de uma residência, prédio, empresas, escolas e outros locais. Ela é extremamente simples, mas é limitada ao espaço físico. A WLAN é a versão alternativa, sendo wireless.

- MAN (Metropolitan Area Network): Essa rede conecta múltiplas redes locais, sendo possível, por exemplo, uma empresa com dois escritórios terem a capacidade de trocar dados entre si. WMAN é uma alternativa Wireless.

- RAN (Regional Area Network): RAN é a rede utilizada para conectar os computadores de uma selecionada região geográfica. Esse tipo de rede tem como característica ter alta velocidade graças aos cabos de fibra ótica.

- WAN (Wide Area Network): A rede WAN abrange áreas intercontinentais. Um grande exemplo desse tipo de rede é a Internet, onde usuários do mundo todo se comunicam. Existe também a WWAN, que é wireless.

- PAN (Personal Area Network): É uma rede bastante limitada de curto alcance que faz a conexão entre dispositivos. Um exemplo dessa rede é o Bluetooth.

- SAN (Storage Area Network): A SAN serve exclusivamente para armazenar dados em um servidor através de computadores, sendo limitado a isso.

- CAN (Campus Area Network): CAN é a rede usada para conectar computadores de residências ou prédios diferentes. Bastante comum em universidades e industrias.
