Software de Rede

27 de Abril, 2017.

Software de rede no sentido mais básico, é um software que facilita, melhora ou interage com uma rede de computadores. É um tipo de software de rede onde permite que os computadores se comuniquem uns com os outros, enquanto outro tipo de software de rede fornece aos usuários acesso a programas compartilhados. O software de rede é um componente chave de redes de computadores de hoje em dia, incluindo a Internet. Alguns softwares de rede podem executar tarefas de contabilidade, enquanto outros criam um caminho de comunicação entre computadores, redes ou usuários individuais. Alguns desses programas podem armazenar grandes quantidades de dados e distribuí-los para os usuários ou para outros programas. Quando usado corretamente, o software de rede pode aumentar a produtividade, reduzir a pressão sobre a rede de computadores e até mesmo fornecer maneiras mais eficientes para que os usuários compartilhem informações e se comuniquem. Software de rede é o que faz a maioria dos sistemas de computadores funcionarem, e é um componente essencial do ambiente de computação.

Protocolos

Protocolo é a "língua" dos computadores, ou seja, uma espécie de idioma que segue normas e padrões determinados. É através dos protocolos que é possível a comunicação entre um ou mais computadores. Os protocolos de rede nasceram da necessidade de conectar equipamentos de fornecedores distintos, executando sistemas distintos, sem ter que escrever a cada caso programas específicos. Ambos os computadores devem estar configurados com os mesmos parâmetros e obedecer aos mesmos padrões para que a comunicação possa ser realizada sem erros. Existem diversos tipos de protocolos de rede, variando de acordo com o serviço a ser utilizado. De maneira geral há dois tipos de protocolos: Abertos e Proprietários ou Específicos. Os protocolos Abertos são os protocolos padrões da internet. Este podem comunicar com outros protocolos que utilizam o mesmo padrão de protocolo. Um exemplo seria o TCP/IP, pois ele pode comunicar com várias plataformas como Windows, Linux, Mac e outros. Já os protocolos Proprietários são feitos para ambiente específicos (daí o seu nome), pois ele apenas pode comunicar com uma plataforma padrão. Exemplos desse tipo de protocolo: IPX/SPX, NETBIOS e outros. São exemplos de protocolos de rede: IP (Internet Protocol), DHCP (Dynamic Host Configuration Protocol). Como último item a ser destacado, pode-se falar da Hierarquia de Protocolos, de forma que a maioria das redes é organizada como uma pilha de camadas colocadas umas sobre as outras. Cada camada oferece alguns serviços para as camadas superiores escondendo os detalhes de implementação desses serviços. A camada n de uma máquina se comunica com a camada n de outra máquina. Um conjunto de camadas e protocolos é chamado de arquitetura de rede.
